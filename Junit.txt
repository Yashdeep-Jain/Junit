UNIT TESTING ->

Testing specific units of code independently.

Testing set of units -> integration testing.

Advantages of unit testing ->

-> Find bugs early
-> Easy to fix bugs
-> Reduce the cost and time


How to perform unit Test ->

-> Junit -> most popular framework for unit testing 
-> Junit-4
-> Junit-5
-> Then what is Mockito ( Mocking framework)
   (Mock fake repo or object)




JUNIT 5 composed of several different modules ->

-> Junit platform (TestEngine API, Responsible for launching testing framework on JVM)
-> Junit Jupiter (New annotations, programming and extension model for writing tests, TestEngine implementation)
-> Junit Vintage (Provider support for Junit 3,4 backend compatibility)
