UNIT TESTING ->

Testing specific units of code independently.

Testing set of units -> integration testing.

Advantages of unit testing ->

-> Find bugs early
-> Easy to fix bugs
-> Reduce the cost and time

-> Junit -> most popular framework for unit testing
